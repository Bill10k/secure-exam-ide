# Secure Offline Programming Exam IDE

A secure IDE designed for offline programming examinations, built with Tauri, React, Monaco Editor, and FastAPI.

## Project Structure

- `desktop-app/`: Tauri + React desktop application.
- `backend-server/`: FastAPI server for LTI integration and APIs.
- `docs/`: Project documentation, diagrams, and reports.
- `scripts/`: Utility scripts for development and deployment.
